# FAQ

## Why isn’t my mega menu showing?

- Make sure the menu item label matches the Mega Menu title exactly.
- Confirm the menu is assigned to the **Primary** location.
- Mega Menus only attach to **top-level** items.

## Does this work with Navigation blocks?

Yes. Use a Navigation block item whose label matches the Mega Menu title.

## Can I disable the default styling?

Yes. Turn off **Enable Default Styling** in settings and use your theme styles.
