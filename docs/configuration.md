# Configuration

Settings are available in **Appearance → BB Mega Menu** (or via the Better Builds hub).

## Layout

- **Header / Nav Height Offset**: Offset for the mega menu panel (e.g., `60px` or `4rem`).
- **Panel Padding**: Inner padding for the panel.
- **Z-index**: Stacking order for the panel.

## Appearance

- **Enable Default Styling**: Use built-in styling for caret and panel.
- **Panel Background**: Background color for the panel.
- **Panel Shadow**: Shadow preset for the panel.

## Behavior

- **Transition Speed (ms)**: Duration of the open/close transition.
