# Usage

## 1) Create a Mega Menu

1. Go to **Appearance → Mega Menus**.
2. Click **Add New**.
3. Give it a title (this must match the menu item label you will use).
4. Add your layout/content in the editor.
5. Publish.

## 2) Attach to a classic menu item

1. Open **Appearance → Menus**.
2. Ensure your menu is assigned to the **Primary** location.
3. Add or edit a top-level menu item.
4. Set the label to match your Mega Menu title exactly.
5. Save the menu.

## 3) Attach to a Navigation block

1. Add a **Navigation** block to your header or template.
2. Create a top-level link with a label matching your Mega Menu title.
3. Save your changes.

## Notes

- Mega Menus only attach to **top-level** items.
- The plugin targets the `primary` menu location for classic menus.
