# BB Mega Menu

BB Mega Menu adds a dedicated Mega Menu post type and a simple settings panel so you can build rich dropdowns without custom code.

## What it does

- Create Mega Menus using the WordPress editor.
- Attach Mega Menus to classic menus or Navigation blocks.
- Style with built-in defaults or your theme styles.
