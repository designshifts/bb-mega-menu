# Troubleshooting

## Menu item renders but no panel

- Confirm the Mega Menu post is **published**.
- Ensure the menu item label matches the Mega Menu title exactly.

## Panel shows but styling looks wrong

- Clear cache/minify plugins.
- Disable theme menu overrides temporarily to verify conflicts.
- Check that `bb-mega-menu-base` is loading.

## Navigation block doesn’t open the panel

- Make sure the Navigation item is top-level.
- The label must match the Mega Menu title.
